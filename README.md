\# Pet Shop Management System 🐾



A desktop-based management application built with \*\*C# WinForms\*\* and \*\*SQL Server\*\*. This system is designed to streamline operations for a pet store, including inventory, employee, and customer management.



\## ✨ Features

\* \*\*Admin Dashboard:\*\* Manage employees and view overall shop statistics.

\* \*\*Inventory Management:\*\* Add, update, and track available pets.

\* \*\*Customer Management:\*\* Handle customer registrations and service requests.

\* \*\*Secure Database:\*\* Uses a global configuration for database connectivity.



\## 🛠️ Tech Stack

\* \*\*Language:\*\* C#

\* \*\*Framework:\*\* .NET Framework (WinForms)

\* \*\*Database:\*\* Microsoft SQL Server



\## 🚀 How to Run

1\. Clone the repository.

2\. Open the `.sln` file in \*\*Visual Studio\*\*.

3\. Create a local database and run the provided SQL scripts.

4\. Update your server details in `App.config` (refer to `App.config.example`).

5\. Build and Run the project.
