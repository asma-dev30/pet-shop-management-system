# Pet Shop Management System

## Technologies
- C#
- WinForms
- SQL Server
## Description
A desktop-based application to manage pet shop operations including employees, customers, and products. Implemented full CRUD functionality with a user-friendly interface.
## Features
- Employee management (Add, Edit, Delete, View)
- Customer management
- Product management
- Database integration using SQL Server
- Input validation and error handling
